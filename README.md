# AtCoder

## Quick-links
- [AtCoder](https://atcoder.jp)
- [AtCoder Problems Progress](https://kenkoooo.com/atcoder/#/table/yifanwu)

## Requirements

- gcc 9
- pipenv

## Setup

```sh
$ pipenv install
$ pipenv shell
```

## Usage

### New contest

```sh
$ pipenv run at-gen {contest_id}
```

### Compile & test single task

```sh
$ pipenv run test contests/abc086/D
```
